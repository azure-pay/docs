## Checkout statuses

| Status | Meaning |
| --- | --- |
| `CREATED` | Session created; payment not started. |
| `PENDING_GATES` | Terms or another gate must be completed first. |
| `CHECKOUT_IN_PROGRESS` | Hosted payment form is active or payment is processing. |
| `CAPTURED` | Payment captured. |
| `FAILED` | Payment failed. |
| `EXPIRED` | Session expired. |
| `REVERSED` | Payment reversed. |

The widget may expose lowercase status values in browser-safe status responses.

## Widget error codes

| Code | Meaning | What to do |
| --- | --- | --- |
| `session_not_found` | Session ID is invalid. | Create a new session or check ID. |
| `session_expired` | Checkout expired. | Let customer restart checkout. |
| `payment_failed` | Payment failed. | Show retry or alternate payment path. |
| `frame_load_failed` | Hosted card frame did not load. | Check browser console, CSP, network, and acquirer availability. |
| `network_error` | Browser could not reach VoucherPay. | Retry, check connectivity, inspect API base URL. |
| `poll_timeout` | No final state before timeout. | Reconcile by session ID server-side. |
| `terms_acceptance_failed` | Terms acceptance failed. | Refresh status and retry. |
| `start_payment_failed` | Hosted payment could not start. | Check gates, merchant provisioning, and acquirer errors. |
| `unknown` | Unexpected error. | Capture logs and contact support. |

## API errors

| Error | Meaning | Fix |
| --- | --- | --- |
| `api_key_required` | Missing bearer key. | Add `Authorization: Bearer ...`. |
| `api_key_invalid` | Key malformed, missing, or revoked. | Issue a new API key. |
| `merchant_not_found` | Key points to missing merchant. | Contact VoucherPay support. |
| `acquirer_not_configured` | API lacks acquirer credentials. | Environment issue. |
| `acquirer_not_provisioned` | Merchant not provisioned. | Complete onboarding/provisioning. |
| `redirect_origin_not_configured` | No allowed origins configured. | Add storefront origin. |
| `redirect_origin_not_allowed` | Redirect origin not allowed. | Add/correct origin. |
| `merchant_terms_not_published` | Terms required but no published terms. | Contact VoucherPay support. |
| `kyc_not_implemented` | Cardholder KYC is enabled but not live. | Disable cardholder KYC until available. |
| `acquirer_error` | Acquirer rejected or failed request. | Inspect response and retry only when safe. |
